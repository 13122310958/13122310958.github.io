Ionic Input Bug 1 - need to put the input in ion-list>ion-item 

<ion-grid>
<ion-row>
</ion-col>
<ion-item>
                <ion-label color="black" >test1</ion-label>
                <ion-input type="number"  text-right placeholder="0" name="a" [(ngModel)]="data.a" pattern="\d*">
                  </ion-input>
              </ion-item>
</ion-item>
</ion-col>
</ion-row>
<ion-row>
<ion-col>
<ion-item>
                <ion-label color="black" >can't go this input in ios</ion-label>
                <ion-input type="number"  text-right placeholder="0" name="b" [(ngModel)]="data.b" pattern="\d*">
                  </ion-input>

              </ion-item>
</ion-item>
</ion-col>
</ion-row>
</ion-grid>

在这样的结构中， ion-input 无法被点击，跳转到下一个input输入点。除非是键盘按 next 才可以。

之后改为：

<ion-list>
<ion-item>
 <ion-label color="black" >test1</ion-label>
                <ion-input type="number"  text-right placeholder="0" name="a" [(ngModel)]="data.a" pattern="\d*">
                  </ion-input>
              </ion-item>
  </ion-item>
         <ion-item>
                <ion-label color="black" >can't go this input in ios</ion-label>
                <ion-input type="number"  text-right placeholder="0" name="b" [(ngModel)]="data.b" pattern="\d*">
                  </ion-input>

              </ion-item>
</ion-item>
</ion-list>

就可以了。 就是要放在 list 里面才可以。放在其它的，都不行。

以此为鉴
